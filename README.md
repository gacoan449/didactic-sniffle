# 🌾 Petani Desa Berkah

Aplikasi supermarket digital berbasis Flutter.

## Visi

Membantu masyarakat desa berbelanja kebutuhan dengan mudah dan terjangkau.

## Fitur

- Login Pembeli
- Login Kasir
- Login Pemilik
- Belanja Online
- QRIS
- Saldo
- Voucher
- Chat
- Voice Call
- Tracking Kurir
- Multi Cabang
- Dashboard Kasir
- Dashboard Pemilik
- AI Customer Service

## Teknologi

- Flutter
- Firebase
- Firestore
- Firebase Auth
- Firebase Storage
- Firebase Messaging
- Google Maps
